SEP10-Project

I learned how to link different files together through the IDE and github with a partner, while consecutively commiting, pushing, and pulling to finish the project.

Content: From the first day to now, Mr.Mueller was expecting us to make a website using a tool and boostrap about our topic.

Challenge: My project was about Medical Practice. I thought of a cool idea, like making an invention using A-frame. Although I wanted to keep it simple and easy, it turned out to be much harder. I had some difficulties with making the website by myself and making the tool due to not knowing how to make it because I had to know a little about geometry first from some parts like rotation or position.

Reflection: Overall, this project has helped me to understand how to create a website using a new tool and bootstrap even more. This has also helped me with coding with my IDE by practicing on how to write different things. It has also helped me with understanding github easier and better. Organization was really a key part in this project due to how much information we needed to have and remember. I also had help from our classmates when I didn't know how to solve something. I find the next step into SEP is to learn Javascript to add the things I want into my project that I couldn't before.
