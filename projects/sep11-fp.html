SEP11 Freedom Project

I learned how to create a game using Kaboom.js and I made a game like Mario.

Content: At the start of the year, Mr.Mueller gave us some tools and asked us to create a project out of it using these tools. I used Kaboom.js to make a Mario like game.

Challenge: My project was making a Mario like game. I was always procrastinating my project because I believed it was easy, but boy did I know how hard it was going to be. I had to ask a lot of friends and go to tutoring for help from Mr.Mueller and seniors to make sure my game was working. One of the biggest challenges was making my level because it wasn't working and I had to find out why. I found out why after looking at the other levels and finding out that I had to use a comma after every row of the level.

Reflection: Overall, this project has helped me to understand how to make a game with Kaboom.js. This has also helped me with coding with my IDE by practicing on how to use different lines of code and how to integrate P5JS into my Kaboom.js. Not procrastinating was really a key part in this project due to the lack of diligence in my work. Something that I'm going to work on next year is to actually be diligent during all my freedom projects.
